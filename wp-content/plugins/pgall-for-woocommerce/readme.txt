== Description ==

워드프레스 쇼핑몰 전문 코드엠샵에서 제공하는 PGALL 우커머스 결제 플러그인은 쇼핑몰 개발 및 운영자를 위한 결제 시스템으로, 설치, 연동, 운영, 업데이트가 쉽고 간편하도록 제작된 대한민국 온라인 결제 시스템 입니다.

결제, 지금까지 단순히 결제로만 이용하고 계셨다면, 쇼핑몰 운영자에게 꼭 필요한 ‘돈버는 쇼핑몰 결제' 를 통해 대박, 성공한 다수의 쇼핑몰들의 비밀을 확인 해 보세요.
쇼핑몰의 매출이 바로 달라집니다.

[기본기능]

- 일반결제, 간편결제, 정기결제 지원
- 심플페이 : 블로그 결제, 일반 페이지 결제, 개인 결제 페이지 지원
- 하나의 PG 플러그인으로 다수의 PG 이용 가능 : KG 이니시스, NHN KCP, LG 유플러스, 나이스정보통신, 카카오페이, NHN 페이코 결제
- 신용카드, 실시간계좌이체, 가상계좌, 휴대폰 소액결제, Kpay, 에스크로, 삼성페이 일반결제 및 간편결제
- 정기결제 : 카카오페이 정기결제, 나이스페이 정기결제

[운영기능]

- 쇼핑몰 매출 통계 : 우커머스의 주문 상태에 따른 일간, 주간, 월간, 년간, 요일별, 시간대별 매출
- 우커머스 교환, 반품 기능 제공
- 시스템 연동의 신용카드 전체환불, 부분환불 기능 제공
- 결제 통계 : 결제 시도, 성공, 포기, 실패건에 대한 분석
- 결제 성공 및 결제 실패 디바이스 (PC, 모바일) / 결제 수단별 분석
- 코드엠샵 네이버페이 플러그인 이용 시, 네이버페이 매출 통계 분석 함께 제공
- 결제 완료 시 리뷰 및 구매평, 별점 자동 등록
- 블로그, 페이지 컨텐츠 작성 후 즉시 결제 기능 제공
- 안전하고 편리한 개인 결제 기능 제공
- 구매자의 내계정에서 결제 영수증 기능 제공

[커스텀 기능]

- 결제창 언어설정 (한국어, 영어, 일어, 중국어)
- 테마에 어울리는 결제창 색상 변경
- 결제창에 쇼핑몰 로고 삽입
- 결제 로딩 이미지 삽입
- 심플페이 숏코드 제공으로 사이트에 적합한 결제창 제작 기능 제공

결제 플러그인 꼭 개발자만 설치, 연동 할 수 있는 것은 아니예요.
코드엠샵에서 개발된 우커머스 결제 플러그인 - PGALL 은 누구나 1분안에 결제 플러그인 설치 및 연동, 설정을 할 수 있도록 만들어진 플러그인 입니다.

대한민국에 우커머스가 상륙하기 이전 부터 개발, 운영되었던 결제 플러그인의 널리지를 지금 경험하세요.

<a href="https://manual.codemshop.com/pgall-for-woocommerce/">매뉴얼 사이트</a>에서 영상 및 텍스트 가이드를 확인 하실 수 있습니다.
플러그인에 대한 문의사항은 <a href="https://www.pgall.co.kr/forums/forum/pgall/">PGALL 포럼</a>을 이용해주세요.

== 플러그인 설치안내 ==
1. 워드프레스 플러그인 화면에서 "PGALL 워드프레스 결제"를 검색하여 설치하거나, 다운로드 받은 파일을 /wp-content/plugins/pgall-for-woocommerce 디렉토리에 업로드하여 설치합니다.
2. PGALL 워드프레스 결제 플러그인을 활성화합니다.
3. PGALL 메뉴에서 플러그인 설정을 진행합니다.

== Frequently Asked Questions ==

= 기존에 코드엠샵 결제 플러그인을 이용하고 있습니다. 새로운 플러그인을 사용하려면 어떻게 해야하나요? =
결제를 비롯한 우커머스 주문연동 및 교환 반품 기능, 매출, 결제 통계를 이용하기 위해서는
1) 사용중이신 결제 플러그인 비활성화를 진행 해 주세요.
2) PGALL 워드프레스 결제 플러그인을 새롭게 다운로드 후, 신규 플러그인의 활성화를 진행합니다.
3) 기존에 발급 받은 상점아이디 (또는 상점키파일) 등록 후 결제 설정을 진행하시면 바로 이용하실 수 있습니다.


= 현재 타사의 결제 플러그인을 이용하고 있는데, 사용이 가능한가요? =
1) 결제 대행사의 고객센터를 통해 상점아이디 (또는 상점키파일)을 코드엠샵으로 변경 해 주세요.
2) 사용중이신 결제 플러그인 비활성화를 진행 해 주세요.
3) PGALL 결제 플러그인을 새롭게 다운로드 후, 신규 플러그인의 활성화를 진행합니다.
4) 새로 발급 받은 상점아이디 (또는 상점키파일) 등록 후 결제 설정을 진행하시면 바로 이용할 수 있습니다.


= 일반결제, 카카오결제, 페이코 등 다양한 결제를 이용하고 싶습니다. 가능한가요? 비용은 어떻게 되나요? =
PGALL 결제 플러그인 하나로 다양한 결제 대행사의 PG 를 이용할 수 있습니다. 물론 여러 회사를 이용시에도 결제 플러그인 사용은 무료 입니다.
1) 이용을 원하는 결제 대행사의 온라인 신청을 진행 해 주세요. 온라인 신청 후 결제 대행사 신규가입팀에서 귀사의 담당자에게 직접 연락을 드립니다.
2) PGALL 결제 플러그인을 다운로드 후 플러그인 활성화를 진행 해 주세요.
3) 결제 대행사로 부터 전달 받은 상점아이디 (또는 상점키파일) 등록 후 결제 설정을 진행 해 주세요.
4) 카드사 심사 완료 후 쇼핑몰에서 다양한 결제 대행사의 PG 를 이용할 수 있습니다.

그외 궁금한 점은  PGALL 포럼 사이트를 이용 해 주세요.
